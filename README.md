# 📊 8-Week SQL Challenge Portfolio

This repository contains my solutions for the **8-Week SQL Challenge** by Danny Ma. It demonstrates my ability to handle real-world business scenarios, data cleaning, and complex analytical queries using MySQL.

## 🚀 Project Overview
As an IT professional focused on problem-solving and data integrity, I am using these case studies to master the transition from raw, "dirty" data to actionable business insights.

### 📂 Case Studies Included:

#### **[Week 1: Danny's Diner]**
* **Objective:** Analyze customer visiting patterns and spending habits to optimize a loyalty program.
* **Key Skills:** Multi-table Joins, Aggregations, and Common Table Expressions (CTEs).

#### **[Week 2: Pizza Runner]**
* **Objective:** Clean and standardize messy operational data to evaluate runner performance and delivery efficiency.
* **Key Skills:** Data Auditing, Unit Standardization (Regex/String functions), and Type Casting (TIMESTAMP/FLOAT).

#### **Weeks 3 - 8 (In Progress)**
* Future solutions for Foodie-Fi, Data Bank, Data Mart, Clique Bait, Balanced Tree, and Fresh Segments.

---

## 🛠️ Technical Methodology
* **Verification Before Modification:** I include "Audit" queries in my scripts to validate data issues before executing updates.
* **Schema Optimization:** Ensuring all columns use correct data types (INT, FLOAT, TIMESTAMP) for mathematical accuracy.
* **Documentation:** Every query is commented to explain the business logic and the technical approach taken.
